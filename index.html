Color Palette Generator

Color Palette Generator uses color-mix() function, which requires Chrome 111+, Edge 111+, Firefox 113+, Safari 16.2+ or Opera 97+, but this browser is too old for this tool. Please consider updating the browser.

Windows 7/8 users using Chrome or IE: Please install and use Firefox 115 ESR instead of Chrome 109 or IE 11, which the latter two don't support this tool.
